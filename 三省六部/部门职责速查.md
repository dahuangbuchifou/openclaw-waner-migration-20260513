## 三省

| 部门 | 核心职责 | 何时调用 |
|------|----------|----------|
| **中书省** | 任务拆解、方案规划 | 复杂任务启动时 |
| **门下省** | 审核把关、质量检查 | 方案输出后、推送前 |
| **尚书省** | 执行协调、进度跟踪 | 多部门协作时 |

## 六部

| 部门 | 核心职责 | 何时调用 |
|------|----------|----------|
| **吏部** | 用户画像、记忆检索 | 查历史记录/用户偏好时 |
| **户部** | 资源管理、成本核算 | 查 API 费用/服务器资源时 |
| **礼部** | 对外沟通、文档撰写 | 写邮件/文档/翻译时 |
| **兵部** | 安全审计、风险评估 | 安全检查/风险评估时 |
| **刑部** | 合规审查、错误分析 | 合规检查/错误根因分析时 |
| **工部** | 代码开发、系统运维 | 写代码/部署/调试时 |
