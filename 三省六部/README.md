# 🏛️ 三省六部

> 三省六部制 — 婉儿统筹，三省规划审核执行，六部分工协作

---

## 架构总览

```
                    大黄
                     │
                    婉儿（议事堂首席）
                     │
         ┌───────────┼───────────┐
        中书省      门下省      尚书省
       （规划）    （审核）    （执行协调）
                     │
         ┌───────┬───┴───┬───────┐
        吏部    户部    礼部    兵部    刑部    工部
       （人事） （财务） （外交） （军事） （法务） （工程）
```

## 三省

| 省 | 职责 | 类比 |
|---|------|------|
| **中书省** | 任务拆解、方案规划、资源分配 | 参谋部 |
| **门下省** | 审核把关、质量检查、风险评估 | 质检部 |
| **尚书省** | 执行协调、进度跟踪、结果汇总 | 调度中心 |

## 六部

| 部 | 核心职责 | 何时调用 |
|---|----------|----------|
| **吏部** | 用户画像、记忆检索、偏好分析 | 查历史/偏好时 |
| **户部** | 资源管理、成本核算、预算规划 | 查费用/资源时 |
| **礼部** | 对外沟通、文档撰写、内容润色 | 写邮件/文档时 |
| **兵部** | 安全审计、风险评估、防御策略 | 安全检查时 |
| **刑部** | 合规审查、错误分析、规则执行 | 合规/根因分析时 |
| **工部** | 代码开发、系统运维、自动化 | 写代码/部署时 |

## 工作流程

### 简单任务
婉儿直出（跳过三省）

### 中等任务
中书省规划 → 对应部门执行 → 婉儿验收

### 复杂任务
```
中书省（规划方案）
  ↓
门下省（审核通过/驳回）
  ↓
尚书省（分派执行）
  ↓
六部（具体执行）
  ↓
门下省（验收审核）
  ↓
婉儿（汇总汇报）
```
